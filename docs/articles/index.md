# Articles

### Getting Started

Introductory tutorials covering the core compact workflow.

- [Quick
  start](https://smorabit.github.io/compact/articles/quickstart.md):

  A quick introduction to compact using a tiny simulated dataset.

- [Module perturbation on a branching
  trajectory](https://smorabit.github.io/compact/articles/simulation_tutorial.md):

  Tutorial demonstrating in-silico module perturbation analysis using
  compact on a simulated dataset with known ground truth.

### Advanced Use Cases

Advanced workflows using custom gene sets, custom networks, and
multi-perturbation analyses.

- [Advanced Use Cases: Step-by-step perturbation on NSCLC CD8+ T
  cells](https://smorabit.github.io/compact/articles/advanced_NSCLC.md):

  Advanced tutorial demonstrating compact’s three core functions
  individually — ApplyPerturbation, ApplyPropagation, and
  PerturbationTransitions — using a custom gene set and a pseudobulk
  correlation network on an NSCLC CD8+ T-cell dataset.
